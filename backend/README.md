# My Rafiki - Backend  

---  

### Brief Introduction  
Backend code for the My Rafiki immigration assistant chatbot.  

### Tech stack  
Python  
PostgreSQL  
Flask  

### Usage  
python -m venv iomuc2_env  
cd iomuc2_env/Script  
activate
cd ../../backend  
pip install -r requirements.txt  
python scraper_and_embedder.py  
python embedder.py
python uc_2_login_prod.py
